Notices

  • Connection issues can occur if your root certificates are outdated. Click here for more information.
  • This instance may contain features that aren't in the main repository yet.
  • Opera Mini support is untested but likely unsupported due to how it handles JavaScript.

PolyMSNP is a lightweight web-based client for the MSNP instant messaging protocol.
It allows you to connect to MSNP11-compatible servers through your web browser.

Select an interface to continue:

  • Generic interface for desktops and tablets
  • Generic interface for mobile devices
  • Legacy interface for older browsers (IE9+)

Below are interfaces that are either experimental or no longer maintained:

  • iOS replica interface (requires WebKit/Safari 6+, unmaintained)

You can check out the GitHub repository for more information.
